# 提示词设计指南

写好提示词是生成高质量图像的关键。

## 提示词结构

一个好的提示词通常包含以下要素：

## 1. 主体描述

明确你想要生成的主要对象：

```
// 例子
"一只可爱的柴犬"
"一个未来风格的城市夜景"
```

## 2. 场景/环境

描述对象所在的场景或环境：

```
// 例子
"在樱花树下睡觉"
"站在雪山之巅"
```

## 3. 风格/艺术手法

指定图像的艺术风格：

```
// 常见风格
水彩风格
油画风格
水墨画风格
赛博朋克风格
蒸汽朋克风格
手绘风格
3D渲染风格
```

## 4. 颜色和氛围

描述整体色调和氛围：

```
// 例子
"柔和的粉色调"
"明亮的蓝色光线"
"温暖的黄色光"
"神秘的紫色氛围"
```

## 5. 细节和材质

添加细节描述以增强真实感：

```
// 例子
"毛发蓬松，纹理清晰"
"金属表面有轻微锈迹"
"水面有波纹和反光"
```

## 6. 技术参数

可以添加拍摄或渲染参数：

```
// 例子
"4K 分辨率"
"焦距 50mm"
"浅景深"
```

## 优秀提示词示例

```
"一只可爱的柴犬在樱花树下睡觉，水彩风格，柔和的粉色调，春天的氛围，毛发蓬松，细节丰富，4K 分辨率"
```

## 提示词优化技巧

### 1. 越详细越好

详细的描述会产生更好的结果：

```
// 不好
"狗在树下"

// 好
"一只可爱的柴犬在樱花树下睡觉，水彩风格，柔和的粉色调"
```

### 2. 使用具体词汇

避免模糊词汇：

```
// 不好
"漂亮的花"

// 好
"粉色的樱花，花瓣飘落"
```

### 3. 顺序很重要

通常遵循从主体到背景的顺序：

```
主体 → 动作 → 场景 → 风格 → 细节
```

### 4. 负数提示词

可以使用负数提示词避免不想要的元素：

```
"一只可爱的柴犬在樱花树下睡觉，水彩风格，不要文字，不要多余的装饰"
```

## 常见错误提示词

### 1. 太过模糊

```
// 避免
"一幅漂亮的画"
```

### 2. 包含矛盾元素

```
// 避免
"一只透明的柴犬"
```

### 3. 违规内容

不要包含违规内容，如暴力、色情等。
