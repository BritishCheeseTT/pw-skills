# 垫图准备指南

垫图可以帮助 AI 理解你想要的风格和构图，生成更符合预期的图像。

## 什么是垫图

垫图是一张参考图片，通过它告诉 AI 你想要的：
- 艺术风格
- 构图方式
- 色彩搭配
- 氛围感觉

## 垫图类型

### 1. 风格垫图

用于指定艺术风格：

```
// 例子
水彩画作品 → 生成水彩风格图像
油画作品 → 生成油画风格图像
动漫截图 → 生成动漫风格图像
```

### 2. 构图垫图

用于指定构图方式：

```
// 例子
三分法构图的图片 → AI 学习三分法构图
对称构图 → AI 学习对称布局
```

### 3. 色彩垫图

用于指定色彩方案：

```
// 例子
莫兰迪色系 → 生成低饱和度、柔和色调的图像
高对比度照片 → 生成色彩鲜明的图像
```

## 如何选择垫图

### 1. 选择高质量图片

垫图质量直接影响生成效果：
- 选择高分辨率图片
- 避免模糊、失焦的图片
- 确保主体清晰可见

### 2. 选择风格明显的图片

风格越明显，AI 学习效果越好：
- 水彩画 → 选择典型水彩作品
- 赛博朋克 → 选择典型赛博朋克风格图片

### 3. 避免复杂场景

简单的垫图更容易让 AI 理解：
- 避免多主体、多场景的复杂图片
- 选择单一主体、清晰背景的图片

## 垫图优化

### 1. 裁剪和调整

```
// 调整垫图
- 裁剪到主体区域
- 调整为合适尺寸（建议 1024x1024 或更大）
- 确保长宽比符合目标图像
```

### 2. 去除水印

垫图中的水印可能影响生成效果：
- 使用去水印工具处理
- 或使用无水印图片

### 3. 格式选择

推荐使用以下格式：
- PNG（无损压缩，保留细节）
- JPG（文件小，适合复杂场景）
- WebP（现代格式，压缩率高）

## 垫图使用技巧

### 1. 多张垫图组合

可以同时使用多张垫图：

```
// 风格垫图 + 内容垫图
{
  "referenceImages": [
    "https://example.com/watercolor-style.png",  // 风格参考
    "https://example.com/mountain-scenery.png"   // 内容参考
  ],
  "prompt": "用第一张图的水彩风格绘制第二张图的山景"
}
```

### 2. 权重调整

通过提示词调整垫图权重：

```
// 强调垫图风格
"参考垫图的水彩风格，绘制一座雪山"

// 弱化垫图风格
"稍微参考垫图的风格，绘制一座雪山"
```

### 3. 迭代优化

```
// 第一次生成
使用垫图 A → 生成图像 B

// 第二次生成
使用垫图 B → 生成图像 C（更接近目标）
```

## 注意事项

1. 版权问题：垫图应使用授权素材或原创图片
2. 公网访问：垫图 URL 需要公网可访问
3. 文件大小：垫图文件不宜过大（建议 < 10MB）
4. 格式要求：确保垫图格式被 API 支持（PNG、JPG、WebP）
